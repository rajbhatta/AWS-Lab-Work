# Working with Nano #
- The caret ("^") means CTRL key . So, ^G means CTRL+G

# Writing using Nano #
- Open file using 
```js
sudo nano text.txt
```

- Nano can let you to edit direclty
- write text
```js
 Ctrl+o
 Enter
 Ctrl + x to exit
 ```
